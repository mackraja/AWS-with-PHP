# AWS-with-PHP
You can controle AWS console commands using php

Required :- 
  1.) You need to add .PEM file of AWS in "File\Pem" Folder.
  2.) Add Username
  3.) Add Password
  4.) Add Database Name
  5.) Add Path (Where to save database)
  
 Note : 
  You can play with other commends of AWS console.
